# Image Processing

## Requirements
- python 2.7
- ipython-notebook
- numpy
- matplotlib
- scipy
- pillow

## Outline
- load images with `scipy.ndimage`
- display images with `matplotlib`
- crop images
- index images (e.g. select the red color channel from a color image)
- filter images
